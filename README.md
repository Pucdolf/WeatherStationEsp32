# Weather Station ESP32 🌦️

A modern IoT ecosystem designed for real-time weather monitoring, bridging **C++ edge computing** with a **scalable .NET backend**.

## 📖 Project Overview

This project aims to provide a complete end-to-end solution for weather data collection and visualization. It features a responsive landing page built with modern CSS techniques (Custom Grids, Flexbox) and is planned to include:

- **Edge Computing:** ESP32 firmware written in C++ for sensor data acquisition.
- **Backend:** Robust .NET API for data processing and management.
- **Infrastructure:** Dockerized SQL environment for reliable data persistence.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Vanilla CSS, Rubik font)
- **Planned Backend:** .NET API
- **Planned Hardware:** ESP32 (C++)
- **Database/DevOps:** Docker, SQL Server
